作品设计与实现：

从模块来讲

1、触摸屏的驱动与LVGL移植，我选取的并非百问网官方的显示屏，选取了一个3.5寸的电容触摸屏，驱动IC分别为ST7796U和FT6336U,此处我打算贴一张产品图，基于此编写了两个芯片的驱动代码，并对接了LVGL的显示和触摸

2、移植了暑假参加电赛的无人机赛题时的飞控代码，主要是通信协议的移植，并利用串口做了测试，作用是因为ra8p1按实际来说是应该接到无人机上的，但是我们不会去造一架无人机出来，我移植的代码可以兼容凌霄飞控的通信协议，方便实际的应用

3、ra6m5上的WIFI芯片W800的驱动，最终实现了通过将开发板和PC端连接到同一网段下，开发板能够接收PC端服务器发送过来的数据并显示到屏幕上

4、ra6m5上的QSPI Flash外设的利用，我们利用这个外设以及板上的W25Q64芯片，并结合littlefs建立了一套日志系统，并通过PC端的一个数据发送脚本不断发送上面提到的凌霄飞控的通信消息，能够实现在屏幕上的日志页面显示历史日志消息，这个也是利好于实际应用的，实时记录飞行数据在无人机领域也很重要，并且后续的ra8p1上也可以做同样的日志系统

5、IMU的驱动，主要是解析了六轴IMU的数据，并显示在了屏幕的页面上，IMU是为了小车的定位服务的

6、小车的运动控制：分为了几个部分，首先是相关的硬件配置，包括PWM输出利用了ra6m5的tim0-3，编码器输入利用了tim4-7设置Input中的计数增减方式实现，以及普通的gpio输出；其次是单一电机的控制，利用了编码器测速结合一阶低通滤波器和前馈控制，实现了稳定的单电机闭环速度控制，

接着是建立四轮小车的运动模型，解算运动到特定位置所需要的控制，最后是结合IMU模块和运动模型实现定位控制（这一块我还没做，你先帮我思考实现路径，并思考可能会运用到哪些提高运动控制效果的方法）
